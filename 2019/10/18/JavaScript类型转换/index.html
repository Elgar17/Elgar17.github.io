前言

JavaScript是个很灵活的语言，之所以很灵活是因为它有着很强大的类型转换，你是不是很掌握了类型转换？来几道题试一下自己的水平😏。

不太会？不慌现在看一下还来得及。

JavaScript 类型转换分为类型转换和隐式类型转换。

显示类型

  • Number(mix)：转化成数字
  • parseInt(str,radix)：转化成整数
  • parseFloat(str)：转化成浮点数
  • toString(radix)：转化成字符串，null和undefined没有toString
  • string()：写什么都转化成字符串
  • Boolean()：转化成布尔值

题：将二进制数 10101010 转化成16进值。

隐式类型转换

隐式类型转换发生在进行这些运算的时候。

  • isNaN(x)：里面的参数放在Number(x)中与NaN比较并且返回结果。

  • ++/-- +/-：先隐式进行Number()运算

  • +：符号一边有字符串,+号会转化成字符串⭐

  • - * / %：先两边的变量都转化成数字

  • && || ！：非将变量转成布尔值，然后取反。

  • < > >= <= ：数字和字符串比较，转化成数字。两边都字符串比较ASCII码大小。

  • == / !=与=== / !==：== / !=存在隐式类型转换，=== / !==不存在类型转换

知道这些以后，都有自己的转换规则，但是还需要记住一些特殊的,没有规则可循。:undefined == null值为true， NaN == NaN返回false。
