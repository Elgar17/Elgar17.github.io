还不会call&apply&bind的区别？

发表于 2019-11-02 更新于 2020-09-26 阅读次数： Valine：

这篇介绍JavaScript中的call&apply&bind的用法与区别🎂。

call

call方法有两个作用,一是调用函数,二是改变this的指向。关于参数:

  • 参数1: 传入的this
  • 参数2,3,..: 给函数传进参数

语法很简单, 调用函数的例子就不写了👇:

✨应用

应用: 改变函数this指向

应用：子构造函数继承父构造函数的属性

apply

apply函数的使用与call很相似,都可以改变this指向,唯一与call的区别：传参形式不同，第二个参数以数组的方式传递。

bind

会改变this指向,不会调用函数,这一点与appy和bind不同的地方,也是他的两点🌤。

  • 参数1：this
  • 参数2，3,..：传递的参数
  • 返回值：由置定this值和初始化参数改造的原函数拷贝
