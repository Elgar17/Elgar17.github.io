快速入门MFC

（在VC++ 6.0 环境下完成的操作）

1.创建环境

新建一个工程 -> 选择MFC AppWizard[exe] -> 填好工程名称

2.简单画线

  • 通过OnDraw函数画出简单的线
  1. 选择单文件 -> 找到类视图里面的View类的OnDraw -> 找到void CMyMFC2View::OnDraw(CDC* pDC) 函数
  • 在函数体里写入pDC->LineTo(100,200);(运行会出现以(0,0)点为起点(100,200)点为终点的一条线)
  1. 调用pDC->MoveTo(x,y); 函数可以改变线的起点

注意：运行程序后将出现的页面最小化然后回复页面是才出现化的线

3.消息响应机制（实现简单弹出框）

  • 通过OnLButtonDown函数实现鼠标点击事件
  1. 找到View类并点击右键添加一个 windows massage Handlers -> 双击WM-LBVTTONDOWN 点击OK
  2. 找到该函数下的 CMyMFC2View::OnLButtonDown()函数
  3. 添加MessageBox("Hello World!");运行(点击鼠标左键会弹出框)
