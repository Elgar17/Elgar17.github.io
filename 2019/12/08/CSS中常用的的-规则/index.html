Less

Less 是一门预处理语言，它是CSS 的扩展，增加了函数、变量等特征所以很好维护，可以运行在浏览器和Node环境中，因为是预处理语言最终会转化成CSS。

1、 @import

在 CSS 文件中导入另一个CSS文件。

2、 charset

用于提示 css 文件使用的字符串编码方式, 必须放在句首。

3、 @meida 媒体查询
